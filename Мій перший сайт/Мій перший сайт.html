Что мы предлагаем?

Курс «Ядро JavaScript» знакомит с языковым ядром, в том числе:

  • переменные и типы данных;
  • функции и операторы управления ходом программы;
  • сбор данных;
  • регулярные выражения;
  • Основы объектно-ориентированного программирования.

Помимо этого, в рамках курса вы получите основную информацию для веб-программирования с использованием JavaScript, в том числе:

  • доступ к модели DOM для динамической генерации и обновления элементов веб-страницы;
  • отладка и тестирование кода в браузере;
  • использование методологии AJAX;
  • знакомство с библиотекой jQuery;
  • регулярные выражения;
  • особенности разработки серверной части веб-приложения с использованием Node.js.

Ради кого?

  • Курс ориентирован на учеников, которые имеют базовые знания языка разметки гипертекста HTML / HTML5 и разбираются в каскадных назначениях таблиц стилей CSS;
  • Для тех, кто хочет освоить популярный язык программирования JavaScript, без которого не обойтись ни в одном проекте в Интернете и не только!
  • Для тех, кто заинтересован в продолжении обучения в SoftServe IT Academy по следующим направлениям веб-разработки: WebUI (Python, PHP, NodeJS, Ruby), .Net, Java и Go.

Продолжительность, расписание и стоимость курса:

  • Продолжительность курса: 1-1,5 месяца, 12 занятий (2-2,5 часа) + курс предусматривает ежедневную работу по 2-4 часа над задачами и проектом, которые можно выполнять дома или в классах SoftServe IT Academy;
  • Расписание занятий: 2-3 занятия в неделю, помимо обычных занятий, участники курса получают доступ к дистанционным курсам дистанционного обучения SoftServe - HTML / CSS и JavaScript, которые не имеют фиксированного расписания;
